# 快速开始

## 最快路径

1. 安装 [OpenCode](https://opencode.ai)。
2. 安装 OpenChamber CLI（一行命令见[安装](/zh-cn/install/)）。
3. 运行 `openchamber --ui-password be-creative-here`。
4. 打开 CLI 输出的 URL（通常是 `http://localhost:3000`）。
5. 想在手机上使用，启动一个[隧道](/zh-cn/tunnels/)并扫描二维码。

你应该会在浏览器中看到 OpenChamber 会话列表。如果它能加载出来，说明你已经成功运行起来了。

请使用强 UI 密码，尤其是当你打算将实例开放到互联网时。

如果页面无法加载，请查看[问题排查](/zh-cn/troubleshooting/)。

## 我应该使用哪个应用？

- 在以 macOS 为主的日常工作中使用 **桌面**
- 在远程访问和移动端审查时使用 **Web**
- 在代码旁进行会话时使用 **VS Code**
