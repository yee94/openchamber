# OpenChamber 文档

OpenChamber 是围绕 OpenCode（在你终端中运行的 AI 编程智能体）的可视化工作空间。它给你一块屏幕去观察并引导这些工作，而不必停留在命令行里。

使用本文档来：

- 为你的工作方式安装合适的应用
- 安全地将 OpenChamber 开放给远程使用
- 自定义外观并排查常见问题

- [隧道](/zh-cn/tunnels/)

## OpenChamber 用于什么

OpenChamber 面向 AI 编程中那些受益于控制中枢的部分：为会话分支、审查代码差异、管理终端、观察工具进度、运行项目操作，并在智能体工作时让整个面板尽收眼底。
