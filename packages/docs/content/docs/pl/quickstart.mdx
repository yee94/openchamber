# Szybki start

## Najszybsza droga

1. Zainstaluj [OpenCode](https://opencode.ai).
2. Zainstaluj CLI OpenChamber (jednowierszowe polecenie znajdziesz w sekcji [Instalacja](/pl/install/)).
3. Uruchom `openchamber --ui-password be-creative-here`.
4. Otwórz adres URL wypisany przez CLI (zwykle `http://localhost:3000`).
5. Aby korzystać z telefonu, uruchom [tunel](/pl/tunnels/) i zeskanuj kod QR.

W przeglądarce powinna pojawić się lista sesji OpenChamber. Jeśli się załadowała, wszystko działa.

Używaj silnego hasła UI, zwłaszcza jeśli planujesz udostępnić instancję w internecie.

Jeśli strona się nie ładuje, zajrzyj do [Rozwiązywania problemów](/pl/troubleshooting/).

## Której platformy użyć?

- użyj **komputera** do codziennej pracy na macOS
- użyj **przeglądarki** do zdalnego dostępu i przeglądania na telefonie
- użyj **VS Code** do sesji tuż obok Twojego kodu
