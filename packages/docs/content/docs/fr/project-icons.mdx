# Icônes de projet

Une icône de projet permet de distinguer vos projets d’un coup d’œil. OpenChamber essaie d’en trouver une pour vous, et vous pouvez toujours définir la vôtre. Gérez-la dans **Paramètres → Projets**.

## Découverte automatique

Quand vous ajoutez un projet, OpenChamber cherche un fichier `favicon` dedans et l’utilise comme icône du projet. Si votre dépôt fournit déjà un favicon, l’icône apparaît généralement toute seule — rien à faire.

## Définir la vôtre

Ouvrez **Paramètres → Projets** et uploadez une image (PNG, JPEG ou SVG, jusqu’à 5 Mo). Une image personnalisée a priorité sur celle découverte automatiquement. Vous pouvez aussi choisir une couleur à la place, ou retirer l’image pour revenir à la découverte.
