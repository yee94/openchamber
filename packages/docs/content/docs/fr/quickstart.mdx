# Démarrage rapide

## Chemin le plus rapide

1. Installez [OpenCode](https://opencode.ai).
2. Installez la CLI OpenChamber (voir [Installation](/install/) pour la commande en une ligne).
3. Lancez `openchamber --ui-password be-creative-here`.
4. Ouvrez l’URL affichée par la CLI (souvent `http://localhost:3000`).
5. Pour l’utiliser depuis votre téléphone, démarrez un [tunnel](/tunnels/) et scannez le QR code.

Vous devriez voir la liste des sessions OpenChamber dans votre navigateur. Si elle se charge, tout est prêt.

Utilisez un mot de passe UI robuste, surtout si vous prévoyez d’ouvrir l’instance à internet.

Si la page ne se charge pas, consultez le [dépannage](/troubleshooting/).

## Quelle application utiliser ?

- utilisez **desktop** pour le travail quotidien sur macOS
- utilisez **web** pour l’accès distant et la revue depuis votre téléphone
- utilisez **VS Code** pour garder les sessions juste à côté de votre code
