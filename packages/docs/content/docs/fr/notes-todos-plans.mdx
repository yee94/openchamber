# Notes, todos et plans de projet

Chaque projet a son propre espace de brouillon pour les notes, une todo list et les plans enregistrés. Ils appartiennent au projet, pas à une session particulière, donc ils restent disponibles lorsque vous passez d’une session à l’autre. Retrouvez-les dans l’onglet **Contexte** de la barre latérale droite (un onglet dédié sur mobile).

## Notes

Une zone de notes libre pour tout ce que vous voulez retenir sur le projet. Elle s’enregistre toute seule pendant que vous tapez.

## Todos

Une checklist simple. Ajoutez des éléments, cochez-les, réordonnez-les par glisser-déposer et supprimez ceux qui sont terminés.

Chaque todo possède un menu **envoyer** pour la confier à l’agent :

- l’envoyer à la session actuelle
- démarrer une nouvelle session avec elle
- démarrer une nouvelle [session worktree](/worktrees/) avec elle (uniquement si le projet est un dépôt git)

## Plans

Un endroit pour conserver des plans plus longs sous forme de fichiers enregistrés. Vous pouvez :

- importer un plan depuis un fichier Markdown ou texte
- ouvrir un plan pour le lire dans le panneau latéral
- supprimer les plans dont vous n’avez plus besoin

Vous devriez revenir dans l’onglet Contexte avec votre note enregistrée, votre todo cochée ou votre plan listé — c’est le signe que l’action a réussi.
