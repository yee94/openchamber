# Utilisation et quotas

La page Utilisation montre quelle part du forfait de chaque fournisseur vous avez consommée, pour savoir à quel point vous approchez d’une limite. Ouvrez-la dans **Paramètres → Utilisation**.

## Ce que vous voyez

Pour chaque fournisseur connecté, OpenChamber affiche :

- la part de la fenêtre actuelle déjà utilisée, sous forme de barre
- une répartition par modèle
- un indicateur de rythme, pour savoir si vous êtes en voie d’épuiser le quota

Vous pouvez choisir les fournisseurs affichés, et le même résumé est disponible depuis un menu déroulant dans l’en-tête de l’application.

## Fournisseurs pris en charge

L’utilisation fonctionne pour les fournisseurs qui publient un quota, notamment Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud et Wafer.

Un fournisseur n’affiche l’utilisation qu’après connexion sur la page [Fournisseurs](/providers/). Quelques fournisseurs demandent une étape supplémentaire — par exemple, Ollama Cloud lit un fichier de session que vous configurez séparément. Si un fournisseur n’affiche aucune donnée, cela signifie généralement que cet identifiant supplémentaire manque.
