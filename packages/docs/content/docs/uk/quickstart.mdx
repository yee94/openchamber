# Швидкий старт

## Найшвидший шлях

1. Встановіть [OpenCode](https://opencode.ai).
2. Встановіть CLI OpenChamber (команду в один рядок дивіться в [Встановленні](/uk/install/)).
3. Виконайте `openchamber --ui-password be-creative-here`.
4. Відкрийте URL, який вивів CLI (зазвичай `http://localhost:3000`).
5. Щоб користуватися з телефона, запустіть [тунель](/uk/tunnels/) і відскануйте QR-код.

У браузері ви маєте побачити список сесій OpenChamber. Якщо він завантажився — все працює.

Використовуйте надійний пароль UI, особливо якщо плануєте відкривати доступ до інстансу з інтернету.

Якщо сторінка не завантажується, перегляньте [Усунення несправностей](/uk/troubleshooting/).

## Яку платформу обрати?

- обирайте **десктоп** для щоденної роботи на macOS
- обирайте **веб** для віддаленого доступу та перегляду з телефона
- обирайте **VS Code** для сесій поруч із вашим кодом
