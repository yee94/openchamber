# Instalación

Hay tres formas de ejecutar OpenChamber:

- app de escritorio para macOS
- app web alojada por la CLI, que puedes instalar como una app de teléfono (una PWA)
- extensión para VS Code

## Requisito previo

Instala [OpenCode](https://opencode.ai) primero: OpenChamber se ejecuta sobre él.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/openchamber/openchamber/main/scripts/install.sh | bash
openchamber --ui-password be-creative-here
```

Abre la URL que imprime la CLI (normalmente `http://localhost:3000`). Deberías ver la lista de sesiones de OpenChamber. Para tenerlo a mano, usa la opción "Instalar" de tu navegador en la barra de direcciones para añadirlo como una app.

## Escritorio

Descarga la última compilación de escritorio desde la página de releases de GitHub o la página de descargas de OpenChamber. Ábrela e inicia sesión con tu flujo de trabajo habitual de OpenCode.

## VS Code

Instálala desde el VS Code Marketplace e inicia sesión con tu flujo de trabajo habitual de OpenCode. La vista de OpenChamber se abre entonces en la barra lateral.
