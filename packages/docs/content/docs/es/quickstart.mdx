# Inicio rápido

## La vía más rápida

1. Instala [OpenCode](https://opencode.ai).
2. Instala la CLI de OpenChamber (consulta [Instalación](/es/install/) para el comando de una sola línea).
3. Ejecuta `openchamber --ui-password be-creative-here`.
4. Abre la URL que imprime la CLI (normalmente `http://localhost:3000`).
5. Para usarlo desde tu teléfono, inicia un [túnel](/es/tunnels/) y escanea el código QR.

Deberías ver la lista de sesiones de OpenChamber en tu navegador. Si carga, ya está todo en marcha.

Usa una contraseña de UI fuerte, sobre todo si piensas exponer la instancia a internet.

Si la página no carga, consulta [Resolución de problemas](/es/troubleshooting/).

## ¿Qué app debería usar?

- usa **escritorio** para el día a día en macOS
- usa **web** para acceso remoto y revisión desde el teléfono
- usa **VS Code** para sesiones junto a tu código
